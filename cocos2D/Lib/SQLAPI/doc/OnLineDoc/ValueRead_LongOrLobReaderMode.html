SAValueRead::LongOrLobReaderMode

SALongOrLobReaderModes_t  LongOrLobReaderMode( ) const;

Returns Long or Lob data reading mode.

Return values

One of the following SALongOrLobReaderModes_t enum values:

  • SA_LongOrLobReaderDefault    Long or Lob(CLob) data reading mode is default.
  • SA_LongOrLobReaderManual    Long or Lob(CLob) data reading mode is manual.

Remarks

Use LongOrLobReaderMode method to check current object's (usually SAFieldor SAParam object) data reading mode.

To set object's data reading mode call setLongOrLobReaderMode method.
