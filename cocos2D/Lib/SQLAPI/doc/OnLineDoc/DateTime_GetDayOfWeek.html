SADateTime::GetDayOfWeek

int GetDayOfWeek ( ) const;

Return value

The day of the week represented by the value of this SADateTime object.

Remarks

Call this member function to get the day of the week represented by this date/time value.

Valid return values range between 1 and 7, where 1=Sunday, 2=Monday, and so on.

See also

GetYear,GetMonth, GetDay, GetHour, GetMinute, GetSecond, GetDayOfYear, Fraction
